# 华为 AppGallery 应用上架完整指南

## 一、准备工作

### 1. 注册华为开发者账号
- 访问 [华为开发者联盟](https://developer.huawei.com/consumer/cn/)
- 注册账号并完成**实名认证**
  - **个人**：需身份证正反面扫描件
  - **企业**：需营业执照副本扫描件（加盖公章）
- 审核时间：**1-3 个工作日**

### 2. 准备应用材料清单

| 材料 | 规格要求 | 备注 |
|------|---------|------|
| **APK/AAB 包** | 华为要求 APK，建议同时提供 AAB | 需签名 |
| **应用图标** | 512×512 px，PNG 格式，无透明通道 | 圆角华为自动处理 |
| **应用截图** | 至少 3 张，推荐 5-8 张 | 5.5/6.5 寸手机截图，最少 1080×1920 |
| **应用描述** | 中文 + 英文 | 中文 10-4000 字，英文 10-4000 字 |
| **隐私政策 URL** | 必须有可访问的页面 | 可用在线文档生成 |
| **版权声明** | 软件著作权登记证书 | **国内市场必备** |
| **应用分类** | 选择最匹配的一级 + 二级分类 | 影响推荐曝光 |
| **关键词** | 最多 10 个 | 影响搜索排名 |

### 3. 合规准备（中国大陆市场）

| 合规项 | 说明 | 是否必须 |
|--------|------|---------|
| **软著** | 软件著作权登记证书 | ✅ 必须 |
| **ICP 备案** | 应用有联网功能时，域名需备案 | ✅ 必须 |
| **等保备案** | 金融、医疗、教育等特殊行业 | 视行业而定 |
| **APP 备案** | 工信部 APP 备案（2023 年起新规） | ✅ 必须 |

## 二、上架流程

### Step 1：创建项目和应用

1. 登录 [AppGallery Connect](https://developer.huawei.com/consumer/cn/appgallery-connect/)
2. 进入「我的项目」→「创建项目」
3. 填写项目信息：
   - 项目名称（公司内部使用）
   - 选择「应用」类型
4. 进入「应用管理」→「创建应用」
5. 填写应用基本信息：
   - **应用名称**（与软著名称建议保持一致）
   - **包名**（如 `com.example.dinoworld`，唯一不可更改）
   - **分类**
   - **分发区域**（中国市场 / 全球市场）

### Step 2：填写应用信息

#### 2.1 基本信息
- 应用名称、简介、一句话描述
- 分类、子分类
- 关键词（搜索优化）

#### 2.2 图片视频素材
| 素材类型 | 要求 |
|---------|------|
| 应用图标 | 512×512 PNG |
| 手机截图 | 至少 3 张，1080×1920 |
| 平板截图（可选） | 1800×2560 |
| 宣传视频（可选） | MP4，最长 30 秒 |

#### 2.3 应用描述
- **简短描述**：10-30 字，一句话介绍
- **详细描述**：100-4000 字，功能介绍、亮点
- **新版本特性**：更新说明

#### 2.4 本地化（可选）
如果面向多语言市场，需配置各语言版本的：
- 应用名称
- 应用描述
- 截图
- 关键词

### Step 3：配置应用签名

华为要求应用必须签名，有两种方式：

| 签名方式 | 说明 | 推荐度 |
|---------|------|--------|
| **华为应用签名服务** | 华为官方代签，保管签名密钥 | ⭐⭐⭐⭐⭐ 推荐 |
| **自有签名** | 开发者自己签名后上传 | ⭐⭐⭐ |

> 💡 **强烈推荐使用华为应用签名服务**：密钥丢失可找回，自己签名的话密钥丢失应用将无法更新。

### Step 4：上传 APK/AAB

1. 进入「发布管理」→「应用版本」→「新建版本」
2. 上传 APK 文件（支持同时上传 AAB）
3. 选择设备类型：
   - 手机
   - 平板
   - 智慧屏
   - 车机
   - 手表
4. 设置版本号（versionCode）和版本名称（versionName）
5. 填写版本更新说明

### Step 5：配置年龄分级

华为有自动分级工具，也可手动填写问卷：
- 暴力内容
- 不良内容
- 赌博
- 烟酒
- 血腥

根据回答自动评定年龄级别（3+、7+、12+、16+、18+）。

### Step 6：配置定价

| 类型 | 说明 |
|------|------|
| **免费** | 完全免费下载使用 |
| **付费** | 一次性买断，设置价格 |
| **免费 + 内购** | 免费下载，应用内购买虚拟商品 |

> 💰 华为收取内购分成 **30%**（使用华为支付可低至 **15%**）

### Step 7：提交审核

1. 确认所有信息填写完整
2. 点击「提交审核」
3. 等待审核结果

**审核时间：**
- 首次上架：**3-7 个工作日**
- 版本更新：**1-3 个工作日**
- 紧急修复：可申请加急

## 三、审核要点（避坑指南）

### 常见被拒原因

| 审核项 | 常见被拒原因 | 解决方案 |
|--------|------------|---------|
| **内容合规** | 涉政、涉黄、涉暴、侵权 | 确保内容符合中国大陆法规 |
| **功能完整** | 应用崩溃、白屏、无法正常使用 | 全面测试后提交 |
| **隐私合规** | 未明示收集哪些数据、未提供隐私政策 | 完善隐私政策，明确告知用户 |
| **应用分类** | 分类与实际功能不符 | 选择最匹配的分类 |
| **截图真实** | 截图与实际应用界面不一致 | 使用真实截图 |
| **包名冲突** | 与已有应用包名重复 | 使用唯一包名 |
| **软著问题** | 软著名称与应用名称不一致 | 确保名称关联 |
| **权限滥用** | 申请了不必要的敏感权限 | 只申请必需的权限 |
| **APP 备案** | 未完成工信部 APP 备案 | 先完成备案再提交 |

### 审核反馈处理
1. 审核不通过会收到驳回原因
2. 修改后重新提交，**无需重新排队**
3. 如有异议可申请复审

## 四、审核通过后

### 1. 上架发布
- 审核通过后，应用**不会立即上架**
- 需手动点击「发布」按钮
- 可选择：
  - **立即发布**
  - **定时发布**（预约上线时间）

### 2. 灰度发布（可选）
华为支持分阶段发布：
- 先对 10% 用户开放
- 观察崩溃率、评价
- 逐步扩大到 100%

### 3. 上架后运营

| 运营动作 | 说明 |
|---------|------|
| **评论管理** | 及时回复用户评价，提升评分 |
| **版本更新** | 定期更新，保持活跃度 |
| **数据分析** | 通过 AppGallery Connect 查看下载量、日活、留存 |
| **推广活动** | 参与华为应用市场推荐活动 |
| **ASO 优化** | 优化关键词、描述提升搜索排名 |

## 五、常见问题 FAQ

**Q: 上架需要多久？**
> 首次审核通常 **3-7 个工作日**（含软著审核），后续版本更新 **1-3 天**

**Q: 必须做华为适配吗？**
> 不是必须，但建议接入 **HMS Core**（推送、定位、地图、账号登录等），可以获得更多曝光和推荐位

**Q: 国内和海外是同一个审核吗？**
> **不是**。国内市场和中国大陆以外市场是**分开审核**的，审核标准和严格程度不同

**Q: 上架收费吗？**
> 上架免费，但华为会收取**应用内购分成**（通常 30%，华为手机支付可低至 15%）

**Q: 个人开发者可以上架吗？**
> 可以，但**中国大陆市场建议用企业主体**，个人开发者部分类目受限

**Q: 已上架应用被下架怎么办？**
> 1. 查看下架原因
> 2. 按要求整改
> 3. 重新提交审核
> 4. 严重违规可能面临账号封禁

**Q: 华为应用市场和 Google Play 能同时上架吗？**
> 可以，完全独立，互不影响。建议同时准备两个平台的包

## 六、完整流程图

```
注册开发者账号（1-3 天）
    ↓
实名认证（1-3 天）
    ↓
准备材料：软著 + ICP备案 + APP备案（2-4 周，可并行）
    ↓
AppGallery Connect 创建项目和应用
    ↓
填写应用信息 + 上传截图 + 配置签名
    ↓
上传 APK/AAB
    ↓
配置年龄分级 + 定价
    ↓
提交审核（3-7 天）
    ↓
审核通过 → 手动发布（或定时发布）
    ↓
正式上线运营
```

## 七、关键时间线

| 阶段 | 时间 | 可并行 |
|------|------|--------|

| ICP 备案 | 10-20 工作日 | ✅ |

| 首次提交审核 | 3-7 工作日 | — |
| 修改后复审 | 1-3 工作日 | — |

> 💡 **建议提前 1-2 个月准备软著和备案，与应用开发并行进行**
